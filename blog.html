Important FAQ

What is the Documents Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents.The DOM represents the document as nodes and objects.It defines the logical structure of documents and the way a document is accessed and manipulated

How do you select an element from DOM?

We can select an element from dom by using an ID (.getElementById)
Also we can use CSS selector like (.querySelector()) and for multiple elements (.querySelectorAll())

What is event delegation in the context of the DOM ,and why is it useful

Event delegation refers to the process of using event propagation to handle events at a higher level in the DOM than the element on which the event originated.It allows to attach a single event listener for elements that exist now or in the future.

How do you manipulate an elements attribute and style using the Dom
